# 🎓 Lista de Carreras – App Android (Control ListView)

## 📌 Descripción del Proyecto

Esta aplicación móvil fue desarrollada en Android como parte de una práctica académica sobre el desarrollo de Interfaces Gráficas de Usuario (GUI).

La aplicación muestra una lista de carreras universitarias utilizando un componente visual de tipo ListView con un diseño moderno basado en CardView.

Cuando el usuario selecciona una carrera de la lista, la aplicación muestra un mensaje emergente (Toast) indicando cuál carrera fue seleccionada.

Además, la aplicación incluye una interfaz visual organizada con título, lista de carreras y un footer informativo con los desarrolladores del proyecto.

---

## 🎯 Objetivos del Proyecto

- Implementar una interfaz gráfica en Android utilizando XML.
- Utilizar el componente ListView para mostrar listas de datos.
- Manejar eventos de selección de elementos en una lista.
- Mostrar mensajes al usuario mediante Toast.
- Aplicar diseño visual utilizando CardView.
- Organizar correctamente los componentes dentro de la interfaz.

---

## 🧩 Componentes de Interfaz Utilizados

- **TextView:** Para mostrar el título de la aplicación y el contenido del footer.
- **ListView:** Para mostrar la lista de carreras disponibles.
- **CardView:** Para mejorar el diseño visual de cada elemento de la lista.
- **LinearLayout:** Para organizar los elementos dentro de la pantalla.
- **Toast:** Para mostrar mensajes al seleccionar una carrera.
